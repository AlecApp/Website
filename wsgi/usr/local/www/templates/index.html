This website is a showcase for my AWS projects.

This website is running on a single t2.micro EC2 instance in my own AWS cloud. The server is a LAMP server hosting a Flask (Python) application. I built the image myself based on AWS Linux 2.
I chose Flask over Node.js because I wanted to demonstrate knowledge of Python. Below, I've included some live examples to showcase knowledge of AWS, Python, and tech in general.

The source code for each function is linked at the bottom of that function's description.

Please keep in mind that this website is a WIP. Launched 10/29/20. Almost everything demonstrated here (Python, JavaScript, etc.) is self-taught.

Thanks for checking it out.

This function takes an input variable (a year between 1920 - 2017) and queries a DynamoDB table built with data pulled from IMDB.
Uses the boto3 API. Data is returned as a JSON string, which JavaScript parses and sanitizes before presenting it.
The displayed movie is chosen at random from the query results for the input year. (This makes it useful for playing Charades!)
