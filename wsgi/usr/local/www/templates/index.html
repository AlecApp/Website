Welcome to Alec's Website

This website is running on a single t2.micro EC2 instance in my own AWS cloud. The server is a Dockerized Apache + Flask (Python) application with HTML + JavaScript for the frontend. Source Code is available here: GitHub
The server is part of a cloud environment which includes an Aurora Postgres cluster (RDS) and numerous other components. The server and its infrastructure are built via IaC (Terraform) and a CI/CD pipeline (GitHub Actions). Below, I've included some live examples of Python functions demonstrating API calls to the AWS resources.

Please keep in mind that this website is a WIP. Almost everything demonstrated here (Python, Terraform, JavaScript, etc.) is self-taught. Also note that the server only supports HTTP (not HTTPS). If you encounter a 500 error, make sure you're connecting via the correct protocol.

Thanks for checking this out.
